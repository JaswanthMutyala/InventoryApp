Inventory App

A simple and efficient inventory management application built using modern web technologies.  
This app helps you track products, quantities, pricing, and update stock in real-time.

Features

Add, update, and delete items  
Search and filter inventory  
Track stock quantity and price  
Fast, responsive UI  
Data storage (local or database integration depending on your setup)

Tech Stack
**Frontend:** React / Vite (or your framework)
**Backend:** Node.js + Express  
**Database:** MySQL / MongoDB (whichever you are using)  
**Styling:** Tailwind CSS (optional)

Install Dependencies:
npm install

start development server:
npm run dev

Database Setup(SQL):
CREATE DATABASE inventory_app;

DB_HOST=localhost
DB_USER=root
DB_PASS=yourpassword
DB_NAME=inventory_app

Build for production:
npm run build
